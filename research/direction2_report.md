## 1. 衰减曲线分析

### 1.1 信号半衰期

| 体制 | 半衰期（天） | 峰值收益 | 说明 |
|------|--------------|---------|------|
| bull | 20+ | 1.99% | 整个持有期内未衰减到峰值50% |
| neutral | 20+ | 0.22% | 整个持有期内未衰减到峰值50% |
| bear | 20+ | 1.92% | 整个持有期内未衰减到峰值50% |

**结论**: 当前策略的选股质量较高，动量信号在20天持有期内**持续有效**，无需担心信号快速衰减。

### 1.2 半衰期计算方法

**定义**: 信号触发后，累积收益衰减到峰值50%所需的天数

**计算公式**:
```
1. 找到持有期内的峰值收益: peak_ret = max(return[t] for t in 1..20)
2. 计算半衰期阈值: half_threshold = peak_ret * 0.5
3. 找到峰值之后，第一个 return[t] <= half_threshold 的天数 t
4. 半衰期 = t - peak_day
5. 如果20天内未衰减到50%，返回20（未衰减）
```

**数据来源**: 
- 回测窗口: `data/market_regime/backtest_results.json` (36个窗口)
- 价格数据: `data/market_regime/combined_daily.json`
- 计算方法: 对每个窗口的选中ETF，计算持有期内每日累积收益，按体制分层求平均

### B. 半衰期计算方法（详细说明）

**步骤**:
1. 从 `backtest_results.json` 加载36个回测窗口
2. 对每个窗口，使用 `combined_daily.json` 计算选中ETF在持有期（20天）内的每日累积收益
3. 按体制（bull/neutral/bear）分层，计算平均累积收益曲线
4. 对每个体制的曲线，找到峰值收益
5. 计算半衰期：从峰值衰减到50%所需天数
6. 如果20天内未衰减到50%，返回20（未衰减）

**代码示例**:
```python
def calculate_half_life(curve):
    peak_day, peak_ret = max(curve, key=lambda x: x[1])
    half_threshold = peak_ret * 0.5
    
    for day, ret in curve:
        if day > peak_day and ret <= half_threshold:
            return day - peak_day
    
    return 20  # 20天内未衰减到50%
```
