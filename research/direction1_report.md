## 3. 信号定义与方法论

### 3.1 信号计算公式

| 信号 | 计算公式 | 数据来源 |
|------|---------|---------|
| ma20_decline | `ma20_ratio = (站上MA20的ETF数) / (总ETF数) * 100`<br>`信号触发: ma20_ratio < 60` | combined_daily.json |
| momentum_decay | `top5_momentum = 前5强ETF的20日动量均值`<br>`20日动量 = (价格_t / 价格_t-20 - 1) * 100`<br>`信号触发: top5_momentum < 3.0` | combined_daily.json |
| ma60_decline | `ma60_ratio = (站上MA60的ETF数) / (总ETF数) * 100`<br>`信号触发: ma60_ratio < 55` | combined_daily.json |

### 3.2 准确率计算方法

- **提前窗口**: 5天（信号触发后5天内发生体制转换，计为正确）
- **准确率**: `正确次数 / 总转换次数 * 100%`
- **数据来源**: `combined_daily.json` (2001个交易日)
