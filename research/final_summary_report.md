## 方向2：动量信号衰减曲线 (修正后)

### 关键发现

| 指标 | 数值 | 说明 |
|------|------|------|
| 信号半衰期 | **20天** | 整个持有期内未衰减到峰值一半 |
| 最优持有期 | **20天** | 当前持有期合理 |
| 15天持有期 | 4.67% | 略高于20天的4.36%，但交易成本增加 |

### 核心结论

1. **动量信号非常持久** - 半衰期=20天，信号在持有期内持续有效
2. **当前20天持有期合理** - 无需缩短
3. **`direction2_accurate.json` 是正确的** - 已删除错误的 `direction2_analysis_v2.json`

### 半衰期计算方法

**定义**: 信号触发后，累积收益衰减到峰值50%所需的天数

**公式**:
```
1. 找到持有期内的峰值收益: peak_ret = max(return[t] for t in 1..20)
2. 计算半衰期阈值: half_threshold = peak_ret * 0.5
3. 找到峰值之后，第一个 return[t] <= half_threshold 的天数 t
4. 半衰期 = t - peak_day
5. 如果20天内未衰减到50%，返回20（未衰减）
```

## 方向3：风险调整动量排序 (修正后)

### 关键发现

| 排序方法 | 平均收益 (相同36窗口) | 与当前式差异 |
|---------|---------------------|---------------|
| **当前式** | **0.96%** | 基准 |
| Sharpe式 | 0.82% | -14.6% |
| Sortino式 | 0.82% | -14.6% |

### 核心结论

1. **当前排序实际更优** - 风险调整排序并未提升表现（与原始报告相反）
2. **样本对齐后差异变小** - 原始报告的高差异主要来自样本不对齐
3. **暂不建议替换排序** - 当前排序表现更好

### 三种排序公式

| 排序 | 公式 | 说明 |
|------|------|------|
| 当前式 | `ret60 - vol20*0.2` | 简化版 |
| Sharpe式 | `ret60 / vol60` | 风险调整收益 |
| Sortino式 | `ret60 / downside_vol60` | 只考虑下行波动 |
